    2. Starting Out

    Learn how to make lists and tuples. We also learn the basic list manipulation commands.

    • Create a list: [5..13] or [5,10..100] or [ n^2 | n<-[1..100], even n]
    • Create a tuple: (5,12)
    • Key to remember: head, tail, init, last, :, ++
