    Intermediate CS

    Intermediate Computer Science uses the DrRacket environment and the Picturing Programs textbook. Use WeScheme if you cannot install programs, but read the notes on WeScheme because it is different in some ways. Eventually you will want to understand why we learn Racket instead of a more popular language like Python. Extra Info: Online Course The Racket MOOC is a good place to look if you want more detail.
